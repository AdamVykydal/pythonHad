# SNAKE GAME

## Install
### Create and activate venv
`python -m venv venv`

Unix/Macos

`source .venv/bin/activate`

Windows

`.venv\Scripts\activate`

To deactivate venv run `deactivate`

### Install dependencies
`python -m pip install -r requirements.txt`

### Install pre-commit hooks
`pre-commit install`
